Amazon Sales Analysis Dashboard 📊
Project Overview
This project involves a deep dive into an extensive Amazon Sales dataset to transform over 78 Million INR worth of transaction data into a functional, interactive Excel Dashboard.

Handling a dataset of this scale (128,000+ records) required a shift from basic spreadsheet management to robust data architecture, ensuring the final dashboard remained responsive and insight-driven.

🚀 Key Insights Uncovered
Top Performer: T-shirts emerged as the dominant category, driving the highest volume of revenue and transaction frequency.

Geographic Hotspots: Maharashtra stood out as the leading state for sales, identifying it as a primary hub for regional marketing and inventory allocation.

Operational Health: Visualized the split between Merchant and Amazon-fulfilled logistics, allowing for a clear assessment of fulfillment efficiency.

Sales Trends: Identified peak purchasing periods to help forecast future inventory needs.

🛠️ Tools & Technical Workflow
To maintain performance with 128k+ rows, the following technical stack was utilized:

Power Query: Used for ETL (Extract, Transform, Load) processes—cleaning messy headers, handling null values, and formatting currency.

Data Modeling: Established relationships between tables to minimize data redundancy.

Pivot Tables & Calculated Fields: Aggregated massive datasets into digestible metrics without lagging.

Advanced Slicers: Implemented interactive filtering by Date, Region, and Category for a truly dynamic user experience.
